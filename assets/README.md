# Asset Intelligence Engine (AIE)

artifact_id: AIE-001
status: draft
purpose: Teach the institution to analyze photos and media as evidence for trust, emotion, authenticity, and conversion.

## Thesis
Exceptional local-business websites rely on real assets.
The institution should learn which assets increase:
- trust
- community feeling
- premium perception
- conversion

## Inputs
Possible sources:
- Instagram
- website images
- Google Maps photos
- owner uploads
- screenshots

## Pipeline
Asset
→ observation
→ interpretation
→ principle
→ ranking
→ hero selection
→ generation

## Hero Selection Rules
Strong hero candidates often include:
- real people
- product + environment
- emotional expression
- recognizable local context

Weak hero candidates:
- empty scenes
- stock feeling
- unclear subject

## Institutional Rule
Assets are evidence.
Real imagery is a trust mechanism, not decoration.
